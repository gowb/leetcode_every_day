# 使用Docker快速部署LeetCode刷题环境
## 快速开始

1. 克隆项目：

```bash
git clone https://github.com/gowb/leetcode_every_day.git
```

2. 进入项目目录：

```bash
cd leetcode_every_day
```

3. 使用Docker Compose启动服务：

```bash
docker-compose up -d
```

这将会启动项目的Docker容器，使其在后台运行。


## 注意事项

确保你已经安装了Docker和Docker Compose。如果没有，可以参考官方文档进行安装。
